# 🛡 微信撤回对策补丁

适用于 Windows PC 版微信的撤回对策工具，基于 Python。支持微信 2.7.0 至最新版本（含 4.1.x 新版），打包为单文件 exe，无需安装任何依赖。

---

## 功能

- 自动从注册表检测微信安装路径
- 显示当前微信版本及补丁支持状态
- 一键安装撤回对策补丁
- 自动备份原始 DLL，支持一键还原
- 实时日志输出，操作过程透明可见

---

## 使用方法

### 直接使用（推荐）

1. 从 [Releases](../../releases) 下载最新的 `WeChatAntiRevoke.exe`
2. **关闭微信**
3. **右键 → 以管理员身份运行**
4. 确认路径和版本信息无误后，点击「安装补丁」

> 微信更新后需重新安装补丁。还原按钮可随时恢复到补丁前状态。

### 从源码运行

**环境要求：** Python 3.10+，Windows

```bash
pip install customtkinter
python main.py
```

### 打包为 exe

在 Windows 上执行：

```bat
build_exe.bat
```

输出文件：`dist/WeChatAntiRevoke.exe`

## 原理

微信在收到撤回通知时，会调用一个函数删除本地消息记录。工具对该函数入口处的条件跳转指令打补丁，使其直接跳过删除逻辑，消息因此保留在界面中。

### 目标文件

微信 PC 版分两个大版本系，使用不同的 DLL：

| 微信版本 | 目标 DLL | 说明 |
|----------|----------|------|
| 2.7.0 ~ 4.0.x | `WeChatWin.dll` | 传统桌面版 |
| 4.1.0 及以上 | `Weixin.dll` | 新版（应用商店同款） |

工具启动时自动识别安装目录中存在哪个 DLL，无需手动选择。

### 补丁模式

| 模式 | 适用场景 | 原理 |
|------|----------|------|
| 精准匹配 | 特定历史版本 | SHA1 校验原始文件后，在固定偏移处写入新字节 |
| 特征码匹配 | 3.7.0 ~ 最新版 | 搜索特定字节序列（含通配符 `?`），将条件跳转 `je`/`jne` 替换为无条件跳转 `jmp` 或 `nop` |

特征码模式对每个版本区间定义一组搜索/替换字节序列，只要微信未对该位置做结构性重构，新小版本无需重新适配即可复用。

## 注意事项

- 修改系统文件需要**管理员权限**
- 操作前请确保**微信已完全关闭**
- 补丁前会自动备份原始 DLL（`WeChatWin.dll.h.bak` 或 `Weixin.dll.h.bak`），可随时还原
- 微信**自动更新后**需重新安装补丁
- 本工具仅供个人学习研究，请勿用于商业用途
